# Screenshot Instructions

Place your actual screenshots in this folder with the suggested filenames. Recommended steps to capture:

1. Open Kibana at http://localhost:5601 and navigate to the Dashboard you built.
2. Use the OS screenshot tool (PrtSc / Snipping Tool on Windows, Shift-Command-4 on macOS).
3. Save images as PNG and name them as listed in `SUBMISSION.md`.
4. Commit the `artifacts/screenshots/` directory before submission.

Suggested image sizes: 1280x720 or 1920x1080 for clarity.
